## Agent skills

### Issue tracker

Issues live as local markdown files under `.scratch/` in this repo (no remote tracker). See `docs/agents/issue-tracker.md`.

### Triage labels

The five canonical triage role strings, each equal to its role name (no overrides). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout: one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
